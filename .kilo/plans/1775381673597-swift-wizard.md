# تقرير استكشاف مشروع VitaSlims ERP

## نظرة عامة
مشروع VitaSlims ERP هو نظام تخطيط موارد مؤسسية متكامل (Enterprise ERP) مبني بـ Next.js 16 مع Supabase كقاعدة بيانات.

---

## التقنيات المستخدمة

| التقنية | الإصدار/الوصف |
|---------|--------------|
| Frontend | Next.js 16.0.10 |
| UI Framework | React 19.2.0 |
| Styling | Tailwind CSS 4.1.9 + Radix UI |
| Database | Supabase (PostgreSQL) |
| State Management | React Context + Realtime |
| Validation | Zod 3.25.76 |
| Forms | React Hook Form 7.60.0 |
| Testing | Vitest 2.1.9 |
| Analytics | Vercel Analytics + Sentry |

---

## هيكل المشروع الرئيسي

### app/ (50+ صفحة)
- **المحاسبة**: accounting, journal-entries, chart-of-accounts
- **المبيعات**: sales-orders, invoices, customers
- **المشتريات**: purchase-orders, bills, suppliers
- **المخزون**: inventory, products, warehouses
- **الموارد البشرية**: hr/
- **الأصول الثابتة**: fixed-assets
- **المصروفات**: expenses
- **التقارير**: reports/
- **الإدارة**: admin, saas-admin, settings

### lib/ (المكتبات الأساسية)
- **supabase/**: client.ts, server.ts, hooks.ts, middleware.ts
- **permissions-context.tsx**: نظام الصلاحيات والإدوار
- **realtime-provider.tsx**: نظام التحديث الفوري
- **governance-layer.ts**: نظام الحوكمة والامتثال
- **data-visibility-control.ts**: التحكم في عرض البيانات حسب الشركة/الفرع
- **erp-governance-layer.ts**: طبقة الحوكمة المؤسسية

### components/
- **sidebar.tsx**: القائمة الجانبية
- **app-shell.tsx**: هيكل التطبيق الرئيسي
- **ui/**: مكونات Radix UI

---

## الأنظمة الرئيسية

### 1. نظام الصلاحيات (RBAC)
- إدارة الأدوار والصلاحيات
- التحكم في الوصول للصفحات والإجراءات

### 2. نظام الفروع والشركات (Multi-Tenant)
- دعم عدة شركات
- عزل البيانات بين الشركات
- فلترة حسب الفرع

### 3. نظام الإشعارات الفوري (Realtime)
- تحديثات مباشرة من Supabase
- نظام الإشعارات في الوقت الحقيقي

### 4. نظام الحوكمة (Governance)
- التحقق من القيود المحاسبية
- تسجيل عمليات التدقيق

### 5. نظام المخزون
- FIFO Engine
- تتبع تكلفة البضاعة المباعة (COGS)
- المخزون الثالثparty

---

## ملخص

مشروع VitaSlims ERP هو نظام ERP متكامل ومتعدد الشركات مع التركيز على:
- ✅ المحاسبة المالية المتكاملة
- ✅ إدارة المخزون والمبيعات والمشتريات
- ✅ نظام صلاحيات متقدم
- ✅ دعم الوقت الحقيقي
- ✅ دعم اللغة العربية (RTL)
